# REST Assured API Automation Framework


## Overview

Enterprise-level API automation framework developed using REST Assured with Java.

The framework validates REST API services using automated test execution, request and response validation, JSON assertions, and reporting.


## Technology Stack

- Java
- REST Assured
- TestNG
- Maven
- JSON
- GitHub Actions


## Framework Features

- REST API automation
- GET, POST, PUT, DELETE validation
- Request payload validation
- Response status validation
- JSON response validation
- API logging
- Reusable API client design
- Maven execution
- CI/CD integration
